# REST API for an Task Manager Application

* developed this REST API for an Task Manager Application. This API performs all the fundamental CRUD operations of Task Manager Application. 

## Tech Stack

* Java
* Spring Framework
* Spring Boot
* Spring Data JPA
* Spring web
* Hibernate
* MySQL
* Lombok
* Swagger

## Controller

* Employee Controller
* Task Controller
